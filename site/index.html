Privacy first

Useful memory without handing over secrets.

The promise is simple: keep the data local, redact the dangerous parts, and let the user decide what gets stored, exported, or removed.

Default posture

Local-first SQLite storage.

The MVP keeps the source of truth on the developer machine and makes privacy controls visible in the product itself.

  • Secret detection for API keys, tokens, passwords, SSH keys, and cloud credentials.
  • Exclude directories and commands you do not want recorded.
  • Pause and resume recording whenever you need a clean boundary.
  • Export or delete all captured data on demand.
  • Encrypted backup bundles for safer local retention.
